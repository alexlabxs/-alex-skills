# 小点 · 童心系列配图

为 🧸 童心系列（3-6 岁亲子共读）文章生成横版配图。

视觉 IP 是"小点"——一个白色圆点小精灵，用光晕"点亮"故事里的关键瞬间。它不是来干活的，是来陪同孩子发现的。

## 工作流

### 1. 读文章

读故事，标记哪些段落适合用图表达——但不是"解释概念"，而是"让孩子看见一个温暖的瞬间"。

童心系列的配图原则：
- 不画抽象概念（如"分享很重要"）
- 画具体瞬间（如"小兔子把饼干递给小熊时，眼睛亮了一下"）
- 每张图只捕捉一个情绪/动作

### 2. 出 shot list

| 字段 | 说明 |
|------|------|
| 场景 | 故事中的具体段落 |
| 小点动作 | 小点在图中做什么——照亮什么/凑近看什么/在哪里发光 |
| 氛围 | 温暖 / 好奇 / 勇敢 / 安心 |
| 光晕颜色 | 淡金 / 暖橙 / 浅红 / 淡蓝 / 淡紫 |
| 画面描述 | 给生图模型的描述 |

### 3. 生图

用 `baoyu-image-gen` + DashScope，加载本技能参考文件确保风格。
