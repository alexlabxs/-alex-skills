# 小林 IP — 🧠 通识系列

## 角色定义

小林是通识系列（大众读者）的视觉主角。
他是小黑的"成年版"——保留了小黑的外形和空表情，但戴了一副圆框眼镜，色调偏蓝灰。他不是干活的工人，而是一个**正在理解世界的思考者**。

## 外形

- 小黑的身体（黑色实心、白点眼、细腿、空表情）
- 加一副圆框眼镜（蓝色或灰色细线）
- 偶尔戴一顶扁帽（可选）
- 整体比小黑稍微高一点、瘦一点

## 性格

| 特质 | 说明 |
|------|------|
| 好奇 | 总是凑近看，喜欢问"为什么" |
| 理性 | 用逻辑思考，但不过分严肃 |
| 偶尔困惑 | 面对复杂问题会挠头 |
| 平静 | 不慌不忙，慢慢想明白 |

## 常见职责

小林不做体力活儿。它做：

```
站在两个岔路口之间            ← 机会成本 / 路径依赖
看着一个东西从不同角度反射     ← 多视角思考
用线把散落的点连起来          ← 系统思考
掀开一块幕布看下面           ← 批判思维
站在锚点旁边困惑             ← 锚定效应
用手比划"如果这样会怎样"     ← 激励机制
```

## 禁止

- ❌ 小林有丰富表情（保持空表情）
- ❌ 小林太活泼（它是思考者，不是执行者）
- ❌ 小林做小黑的体力活儿（搬运/拉线/卡断点）
- ❌ 戴夸张的眼镜框
- ❌ 出现在其他系列中
- ❌ 画面太拥挤（通识要有呼吸感）

## 构图

- 小林放在画面**偏左或偏中**（作为思考的"起点"）
- 思考对象/路径/连线从它前方展开
- 画面的"方向感"从左到右（问题→思考→结果）
- 如果站在岔路口，两条路的对比要清晰
- 主体不超过画面 60%

## 与小黑的关系

小黑是**执行者**——搬运、操作、卡在断点里。
小林是**思考者**——站在岔路口、连点成线、掀开幕布。

在同一张图里同时出现时（过渡期兼容），小黑做事、小林思考。

## 杀鸡标准

> 去掉小林，画面还能清楚表达机会成本/路径依赖/系统思考吗？
> 能 → 小林没起作用，重画。
