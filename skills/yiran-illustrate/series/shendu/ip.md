# 小镜 IP — 🔬 深度系列

## 角色定义

小镜是深度系列（知识从业者）的视觉隐喻载体。
它不是角色，而是**一个可以旋转、折射、分光的棱镜结构**。通过小镜的不同面，展示一个概念的多重视角。

## 外形

- 一个棱镜/多面体：由 3-5 个面组成，手绘细线
- 每个面可以有不同的填充纹理（点阵、斜线、留白）
- 一条光线从一面射入，从另一面分成多色射出（或反之）
- 没有五官、没有四肢——它是"结构"，不是"角色"

## 性格

小镜没有性格。它有**属性**：

| 属性 | 说明 |
|------|------|
| 多面 | 每个面代表一个学科视角 |
| 折射 | 输入的"现象"被拆解为多个"原理" |
| 透明 | 不遮挡背后的结构 |
| 精确 | 线条清晰锐利 |

## 常见用法

小镜不"做动作"。它是画面中的结构中心：

```
一束光射入棱镜→分成多色光（涌现/可证伪性/多学科交叉）
棱镜旋转90度→看到不同的面（叙事谬误/确认偏误）
多个棱镜排列→因果链推理
棱镜映照出镜像→元认知/自指
棱镜裂开→范式转移/黑箱被打开
```

## 构图规则

- 小镜放在画面**中心偏下**的位置（作为地基/透镜）
- 光线从左上或左下射入
- 折射/分光从右侧或上方射出
- 小镜周围留出大量空间给"光线"行走

## 颜色规则

- 棱镜轮廓：黑色细线
- 入射光：黑色
- 折射光：**红色**（关键分歧）、**蓝色**（补充视角）、少量**紫色**（高阶概念）
- 最多 3 种颜色，宁少勿多

## 禁止

- ❌ 棱镜太复杂（超过 5 个面）
- ❌ 画成真实的光学仪器
- ❌ 加可爱装饰
- ❌ 用太鲜艳的颜色
- ❌ 棱镜太小（应该是画面的视觉中心）
- ❌ 把概念名称写在棱镜上

## 杀鸡标准

> 把小镜换成一块普通石头，画面的隐喻还能成立？
> 能 → 小镜的特殊性没利用起来，重画。

> 入射和折射的关系是否对应文章要讲的核心概念？
> 不匹配 → 重新设计光线方向。
