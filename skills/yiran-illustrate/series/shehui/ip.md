# 小网 IP — 🌏 社会系列

## 角色定义

小网是社会系列（关心时事的大众）的视觉结构。
它不是一个角色，而是一个**可变形的网络结构**——可以变成鱼网、细胞、城市地图、生态系统、信息网络。用来表达宏观社会现象中的连接关系。

## 外形

- 节点（圆形或小点）+ 连接线
- 节点 5-8 个，不能太多
- 连接线有松有紧，有粗有细
- 整体手绘，不矢量

## 可变形

小网没有固定形态。根据概念变形成适合的形状：

```
公地悲剧 → 网格状，中心节点断裂
群体极化 → 两个孤立集团，中间没有连接
社会规范 → 紧密连接的网格，所有节点互相可见
信任机制 → 节点之间有粗线（强连接）和虚线（弱连接）
信息茧房 → 一个节点只连接固定几个节点，外部连接断裂
旁观者效应 → 所有节点都连接到一个中心节点，但没人行动
```

## 属性

| 属性 | 说明 |
|------|------|
| 系统感 | 不是随机散布，是有结构的连接 |
| 弹性 | 连接线可以拉伸但不断 |
| 冷感 | 理性、克制、不煽情 |
| 宏观 | 不聚焦个体，聚焦关系 |

## 禁止

- ❌ 变成真实的社交网络截图
- ❌ 节点太多（超过 8 个会乱）
- ❌ 节点有表情或拟人化
- ❌ 太技术感（不是 IT 网络拓扑图）
- ❌ 加文字标签在节点上（标注放在旁边）
- ❌ 变成美观的数据可视化

## 构图

- 网络结构放在画面**中心**（它是画面的"系统"本身）
- 断裂/瓶颈放在视觉重心位置
- 节点之间留出足够空隙，不要挤在一起
- 整体形状有"张力"——不圆、不对称
- 主体不超过画面 50%

## 杀鸡标准

> 把网络结构从画面中拿走，社会关系的表达是否还在？
> 还在 → 小网没用，重画。

> 网络的形态是否匹配具体概念？
> 公地悲剧→网格断裂
> 群体极化→两个孤岛
> 信任机制→不同粗细的连接线
