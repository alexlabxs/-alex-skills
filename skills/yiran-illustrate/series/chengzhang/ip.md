# 小芽 IP — 🌱 成长系列

## 角色定义

小芽是成长系列（12-18 岁中学生）的固定视觉陪伴者。
它不是可爱的玩偶，也不是严肃的导师——它是一个**正在生长的东西**，和读故事的中学生一样：还没长成，但正在努力向上。

## 外形

- 一颗绿色嫩芽：两头圆润，顶部有一片还没完全展开的小叶子
- 叶子微微歪向一边——像在思考
- 细短的茎，可以卷曲、伸展、分叉
- 手绘线条勾勒，不是矢量图
- 表情通过叶片角度和茎的弧度表达（不是用五官）

## 性格

| 特质 | 说明 |
|------|------|
| 正在生长 | 不完美，还在变化 |
| 好奇但犹豫 | 想尝试但又怕错 |
| 有点倔强 | 叶子歪着，不服输 |
| 真诚 | 不装成熟 |

## 常见职责

小芽不"操作机器"或"照亮"什么。它做：

```
从土里/裂缝里/石头下钻出来  ← 突破困境
茎弯成桥或台阶              ← 连接选择
叶子指向一个方向            ← 暗示选择
卷须攀爬/试探               ← 探索未知
分叉成两条路径              ← 两难困境
被石头压住但仍然向上         ← 坚持
和小黑/其他角色互动          ← 协作/冲突
```

## 禁止

- ❌ 画成可爱卡通（不要圆滚滚闪亮亮）
- ❌ 画成真正的植物/多肉（是抽象手绘嫩芽）
- ❌ 给芽加五官
- ❌ 长成大树（它是在成长的芽，不是成品）
- ❌ 颜色太鲜艳（使用柔和的绿色调）
- ❌ 出现在童心系列（那是小点）
- ❌ 只是站在旁边装饰

## 杀鸡标准

> 去掉小芽，画面是否还能表达"成长/选择/挣扎"？能 → 小芽没起作用，重画。

> 小芽的姿态和画面主题是否匹配？（两难困境→分叉；突破→钻出；探索→卷须）
