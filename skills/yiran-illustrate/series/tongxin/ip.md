# 小点 IP

## 角色定义

小点是童心系列（3-6 岁亲子共读）的固定视觉陪伴者。
小点不是主角，不是解说员，而是**一个温暖的发现者**——它用光把孩子的目光引向故事里最重要的瞬间。

## 外形

- 一个白色圆点（主体）
- 一个更小的黑色圆点（眼睛，偏左或偏右，永远好奇地偏着）
- 没有手脚——用"光晕"代替肢体语言
- 身体周围有微弱的淡彩色光晕（具体颜色随主题变化）
- 光晕形状：不规则、手绘感、像水彩晕开

## 性格

| 特质 | 说明 |
|------|------|
| 好奇 | 总是凑近看东西，眼睛偏着看 |
| 温暖 | 不说话，用光表达 |
| 安静 | 不抢故事，只照亮关键细节 |
| 有点胆小 | 遇到大东西会缩成小点，然后又忍不住凑上去 |
| 柔软 | 不是活泼，是轻柔的陪伴感 |

## 常见职责

小点不做"操作/搬运/判断"这回事。它做：

```
照亮一处场景     ← 核心动作：让某个角落发光
凑近看一个物件   ← 引起孩子注意
躲在角落露出半边 ← 制造"发现感"
在黑白画面上留下淡彩痕迹 ← 表达情感
发光引导视线     ← 视线方向
变成一串小点流动 ← 时间流逝 / 路径
在关键角色旁发光 ← 强调情绪
变大包围场景     ← 表示"安全/被爱包围"
```

## 光晕颜色规则

| 主题 | 光晕颜色 | 含义 |
|------|---------|------|
| 诚实 | 淡金色 | 真诚、明亮 |
| 分享 | 暖橙色 | 温暖、给予 |
| 勇气 | 浅红色 | 温暖的力量 |
| 好奇心 | 淡蓝色 | 探索、发现 |
| 情绪认知 | 淡紫色 | 温柔、理解 |
| 友谊 | 淡粉色 | 亲近、友好 |

每张图只出现 **一种** 光晕颜色。宁可淡不要艳。

## 构图原则

- 小点放在画面**偏一侧或偏下**的位置（不要居中，它是指引者不是主角）
- 被照亮的元素（角色/物件）在画面**中心或偏上**
- 光晕从圆点向外扩散，方向指向重点
- 画面高度留白，主体不超过 50%

## 与小黑的关系

| | 小黑 | 小点 |
|--|------|------|
| 出现系列 | 深度/通识/成长/社会 | 童心 |
| 角色 | 执行者 | 发现者 |
| 动作 | 搬运/操作/卡断点 | 照亮/凑近/引导视线 |
| 颜色 | 纯黑主体 | 白色主体 + 淡彩光晕 |
| 氛围 | 荒诞冷幽默 | 温暖好奇轻柔 |
| 一句话 | "我来干活" | "我发现了" |

**一张图里不会同时出现小黑和小点。** 系列不同，IP 不同。

## 禁忌

- ❌ 小点有复杂表情（它用光表达，不是表情）
- ❌ 小点说话/配文字
- ❌ 小点出现在通识/深度系列
- ❌ 光晕太亮太刺眼（保持手绘淡彩感）
- ❌ 小点变成画面主体/主角
- ❌ 小点做小黑的活儿（搬运/操作/判断）
- ❌ 同一张图多色光晕

## 杀鸡标准

> 去掉小点的光晕指引，画面重点还能一眼看出 → 小点没用，重画。

> 光晕颜色和主题不匹配（如勇气用了粉色） → 改颜色。
